Desired Outcome:

1. A table showing the annual averages of each observed metrics for every city
2. A table showing by how much temperature fluctuates each month from min to min (in%). Take min temperature as the base
3. Table showing the annual maximums of each observed metric for every city
4. Table showing the annual minimums of each observed metric for every city
5. Table showing in which months the annual maximums of each metric were observed in every city (Advanced)
